Ferramenta Comando Descrição
Linux ls Lista de diretórios.
Linux ls -al Lista de diretórios com exibição de arquivos ocultos.
Linux cd dir Muda do diretório atual para o especificado.
Linux cd Muda para o diretório /home (arquivos pessoais).
PHP echo "\033[2J\033[;H"; Sequência de escape ANSI para limpar a tela.
PHP phpinfo(); Exibe informações detalhadas sobre a configuração do PHP em uso.
PHP echo Exibe uma ou mais expressões na saída padrão.
PHP print(); Exibe uma string na saída padrão, retorna 1.
PHP print_r(); Exibe informações básicas sobre uma variável.
PHP var_dump(); Exibe informações detalhadas sobre uma variável.
PHP printf("%s, %d, %.2f", $str, $int, $float); Formata e exibe uma string.
PHP vprintf("%s, %d, %.2f", $array); Formata e exibe uma string a partir de um array(str, int, float).
PHP $other = sprintf("%.2f", $float); Formata uma string e a retorna para outrem.
PHP If ($cond) { ... } Avalia uma condição.
PHP Elseif ($cond) { ... } Fornece uma condição adicional a ser avaliada.
PHP Else { ... } Se todas as condições anteriores forem falsas ...
PHP Switch ($var) { case ...: break; default: ... break;} Avalia uma variável e executa o caso correspondente.
PHP While (condition) { ... $varTemp++} Executa um bloco de código enquanto a condição for verdadeira.
PHP Do { ... $varTemp++} while (condition); Executa o bloco ao menos uma vez, verificando a condição no final.
PHP for ($i = 0; $i !== $n; $i++) { ... continue} Repete um bloco de código com base em condições.
PHP foreach ($array as $item) { ... } Itera sobre os elementos de um array simples.
PHP foreach ($array as $key => $value) { ... } Itera sobre os elementos de um array ou objeto.
PHP If (!empty($variable) { ... }) Verifica se uma variável está vazia (null, false, 0, "", ou não definida).
PHP If (isset($variable) { ... }) Verifica se uma variável está definida e não é null.
PHP $noTags = strip_tags($withTags); Remove todas as tags HTML e PHP de uma string.
PHP $dynamicCall = call_user_func("strip_tags", $withTags); Chama uma função passando-a como string.
PHP $variable = function ($argument) { ... }; Define uma função anônima (Closure), usualmente em runtime.
DEV Branch Uma ramificação a parte do repositório principal de código.
DEV CI/CD (Continuous Integration / Continuous Deployment)
DEV Deploy Movimentar o código para o ambiente de produção.
DEV End-to-End (E2E) Abrange o fluxo completo de uma solução, do início ao fim.
DEV Exception Eventos que precisam ser tratados para fins de evitar falhas.
DEV Hotfix Alterações emergenciais para corrigir erros críticos.
DEV Merge Integrar as alterações de uma branch de volta à branch main.
DEV Mock Versão fictícia de um sistema, protótipos para testes de usabilidade.
DEV Mock Interview Entrevista simulada.
DEV MVP (Minimum Viable Product) Solução funcional e utilizável, passível de feedback de usuários.
DEV POC (Proof of Concept) Demonstração para validar a viabilidade de uma ideia.
DEV Pull Request (PR) Requisitar que as alterações sejam integradas ao repositório principal.
DEV Task Uma unidade de trabalho, uma tarefa, uma funcionalidade, um bug.
